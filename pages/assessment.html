Réviser pour une évaluation

Méthode à suivre

  1. Faire le point (jour 1-2)
  2. Réviser activement (jour 3-4)
  3. Remobiliser ses connaissances (jour 5-6)
Autoévaluation

Exemple de consigne

Préparer l'évaluation sur le chapitre "La vie sociale chez les fourmis".

Exemple de fiche de révision

Révision - La vie sociale des fourmis
Compétences évaluées Connaissances évaluées
(à savoir)
  • S1 - Extraire et organiser des informations.
  • T1 - Mobiliser ses connaissances.
  • S5 - Expliquer des phénomènes (environnement).
  • Les définitions de : caste - colonie - phéromone - trophallaxie.
  • L'organisation d'une colonie.
  • Les castes.
  • La communication dans la colonie.
  • Coopération et division du travail.

Faire le point (jour 1-2)

  • Si possible, se tester sur Éléa sans réviser.
  • Repérer les éléments à connaître (titres, bilans, mots-clés, définitions, schémas).
  • Faire des fiches (une demi-feuille devrait suffire).

Réviser activement (jour 3-4)

  • Réapprendre les leçons.
  • Refaire les activités.
  • Si possible, se tester sur Éléa.

Remobiliser ses connaissances (jour 5-6)

  • Relire une dernière fois les fiches.
  • Si possible, se tester une dernière fois sur Éléa.

Autoévaluation : je suis prêt(e) pour l'évaluation si…

  • Je connais les mots-clés et les définitions sans hésitation.
  • Je peux expliquer les mots-clés avec mes mots.
  • Je sais répondre à des questions courtes.
  • Je sais expliquer avec mes mots le contenu des leçons.
  • Je sais refaire les activités du cours.
  • Mon score sur Éléa est élevé (idéalement, plus de 80 sur 100).
