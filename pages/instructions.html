Apprendre

Définition :
Mémoriser et comprendre pour pouvoir réutiliser plus tard
Réponse attendue :
réponses à des questions
Exemple de consigne :
Apprendre le bilan sur la communication des fourmis.

Calculer

Définition :
Réaliser une opération mathématique
Réponse attendue :
calcul + phrase réponse
Exemple de consigne :
Calculer la vitesse d’une fourmi à partir des données.

Classer

Définition :
Faire des groupes d’objets en se basant sur ce qu’on observe
Réponse attendue :
tableau, listes
Exemple de consigne :
Classer les espèces de fourmis selon leur régime alimentaire.

Comparer

Définition :
Trouver les points communs et les différences entre deux choses
Réponse attendue :
texte / tableau à double entrée
Exemple de consigne :
Comparer les différentes espèces de fourmis.

Compléter

Définition :
Ajouter les informations manquantes dans un document
Réponse attendue :
variable
Exemple de consigne :
Compléter le tableau sur les types de fourmis à l’aide du texte.

Concevoir

Définition :
Inventer une expérience pour résoudre un problème
Réponse attendue :
protocole
Exemple de consigne :
Concevoir une expérience pour savoir ce qu'aiment les fourmis.

Conclure

Définition :
Dire la réponse finale après avoir utilisé les informations
Réponse attendue :
une ou deux phrases
Exemple de consigne :
Conclure sur l’alimentation des fourmis à partir du film.

Convertir

Définition :
Passer d’une unité à une autre
Réponse attendue :
une phrase réponse
Exemple de consigne :
Convertir la distance parcourue par une fourmi en mètres.

Décrire

Définition :
Dire ou écrire ce qu’on voit
Réponse attendue :
quelques phrases
Exemple de consigne :
Décrire le trajet d’une fourmi entre la nourriture et le nid.

Déduire

Définition :
Mettre en relation des informations pour répondre avec logique
Réponse attendue :
quelques phrases
Exemple de consigne :
Déduire ce que fait une fourmi quand elle laisse une trace au sol.

Définir

Définition :
Donner le sens d’un mot ou d’une idée
Réponse attendue :
une ou deux phrases
Exemple de consigne :
Définir « phéromone ».

Déterminer

Définition :
Trouver une information à l’aide d’indices
Réponse attendue :
une ou deux phrases
Exemple de consigne :
Déterminer quel est le meilleur milieu de vie pour les fourmis.

Expliquer

Définition :
Dire ou écrire pourquoi ou comment quelque chose se passe
Réponse attendue :
texte long
Exemple de consigne :
Expliquer comment les fourmis retrouvent leur chemin.

Formuler

Définition :
Exprimer une idée avec ses mots
Réponse attendue :
une phrase
Exemple de consigne :
Formuler une hypothèse sur l’alimentation des fourmis.

Identifier

Définition :
Reconnaître quelque chose en se basant sur des observations
Réponse attendue :
une phrase
Exemple de consigne :
Identifier la reine sur la photo d'une colonie de fourmis.

Indiquer

Définition :
Présenter une information
Réponse attendue :
une phrase
Exemple de consigne :
Indiquer où les œufs sont déposés dans la fourmilière.

Justifier

Définition :
Dire pourquoi une réponse est exacte, avec des preuves
Réponse attendue :
quelques phrases
Exemple de consigne :
Justifier votre réponse en observant ce que font les fourmis.

Légender

Définition :
Nommer les différentes parties d’un schéma ou d’un dessin
Réponse attendue :
un mot par flèche
Exemple de consigne :
Légender le schéma du corps d’une fourmi.

Lister

Définition :
Faire une liste
Réponse attendue :
une liste
Exemple de consigne :
Lister les différents types de fourmis dans une colonie.

Localiser

Définition :
Indiquer où se trouve un lieu ou un objet
Réponse attendue :
variable
Exemple de consigne :
Tracer un graphique montrant le nombre de fourmis sur un an.

Mesurer

Définition :
Faire une mesure avec un instrument adapté
Réponse attendue :
valeur mesurée + unité
Exemple de consigne :
Mesurer la longueur du tunnel creusé par les fourmis.

Modéliser

Définition :
Représenter la réalité de façon simplifiée
Réponse attendue :
maquette
Exemple de consigne :
Modéliser une fourmilière avec une maquette.

Montrer

Définition :
Apporter la preuve que quelque chose est vrai
Réponse attendue :
texte long
Exemple de consigne :
Montrer que les fourmis se repèrent grâce à des phéromones.

Nommer

Définition :
Donner le nom exact de quelque chose
Réponse attendue :
une phrase
Exemple de consigne :
Nommer les parties du corps d’une fourmi.

Placer

Définition :
Mettre un événement ou un lieu sur une carte, une frise
Réponse attendue :
variable
Exemple de consigne :
Placer les étapes de la vie d’une fourmi sur le calendrier.

Présenter

Définition :
Rendre compte d’un travail à l’oral
Réponse attendue :
oral long
Exemple de consigne :
Présenter une espèce de fourmi à la classe.

Proposer

Définition :
Donner une idée pour résoudre un problème
Réponse attendue :
une ou deux phrases
Exemple de consigne :
Proposer une hypothèse sur la communication des fourmis.

Prouver

Définition :
Apporter la preuve que quelque chose est vrai
Réponse attendue :
texte long
Exemple de consigne :
Prouver que c’est la reine qui pond des œufs dans la colonie.

Réaliser (1)

Définition :
Produire un document
Réponse attendue :
variable
Exemple de consigne :
Réaliser un schéma de la fourmilière.

Réaliser (2)

Définition :
Suivre dans l’ordre les étapes d’une expérience
Réponse attendue :
pas de production
Exemple de consigne :
Réaliser l’expérience prouvant que les fourmis détestent le sel.

Rechercher

Définition :
Trouver des informations fiables de façon organisée
Réponse attendue :
variable
Exemple de consigne :
Rechercher sur Internet la structure d'un nid de fourmis.

Rédiger

Définition :
Écrire un texte sur un sujet donné avec un vocabulaire précis
Réponse attendue :
texte long
Exemple de consigne :
Rédiger un texte expliquant la reproduction des fourmis.

Relever

Définition :
Trouver une information précise
Réponse attendue :
variable
Exemple de consigne :
Relever le nombre de fourmis qui passent en 5 minutes.

Relier

Définition :
Faire le lien entre plusieurs idées
Réponse attendue :
une ou deux phrases
Exemple de consigne :
Relier la température au lieu où les fourmis déposent les œufs.

Repérer

Définition :
Utiliser des indices pour placer quelque chose sur un document
Réponse attendue :
variable
Exemple de consigne :
Repérer le chemin emprunté par une fourmi sur une carte.

Reporter

Définition :
Placer des données dans un document
Réponse attendue :
tableau, graphique
Exemple de consigne :
Reporter sur le graphique le nombre de fourmis qui sortent.

Représenter

Définition :
Montrer une idée avec un document différent d’un texte
Réponse attendue :
document de type variable
Exemple de consigne :
Représenter l’organisation de la colonie avec une carte mentale.

Schématiser

Définition :
Représenter la réalité ou un fonctionnement avec des symboles
Réponse attendue :
schéma / schéma fonctionnel
Exemple de consigne :
Schématiser le trajet des œufs dans la fourmilière.

Simuler

Définition :
Reconstituer une expérience à l’aide d’un ordinateur
Réponse attendue :
pas de production
Exemple de consigne :
Simuler la vie d’une colonie avec un logiciel.

Tester

Définition :
Vérifier si une hypothèse est vraie avec une expérience
Réponse attendue :
pas de production
Exemple de consigne :
Tester si les fourmis préfèrent la banane ou le sucre.

Tracer

Définition :
Représenter des données sous forme d’un graphique
Réponse attendue :
graphique
Exemple de consigne :
Tracer un graphique montrant le nombre de fourmis sur le miel.
